# PCPU-NANO1
Simple 1-bit cpu made from transistors
## 1-bit Processor
Minimalistic 1-bit cpu made from npn transistors and resistors

Total Transistor count: 5

I/O pins:

In, Data, Out(not in pcb, led), Jump, Ins

## Instruction Set
- 0 NOP No OPeration
- 1 NADJZ NanD data and in, Jump if Zero
